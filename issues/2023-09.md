# [Notion 공부](https://www.notion.so/)

<div>
  
    1. Notion 시작하기
      1) Notion 홈페이지 접속
      2) 계정 로그인 또는 회원가입
      3) 요금제 - 무료 또는 유료 선택

    2. Notion 게스트 초대하기
      1) 노션 페이지 선택 후 "공유" 클릭
      2) "공유 - 이메일 입력 - 초대" 클릭
      3) 사용권한 설정
        * 전체 허용
        * 편집 허용
        * 댓글 허용
        * 읽기 허용
      4) 공유된 페이지에서 초대됐는지 확인
        * 공유된 페이지 : 초대 멤버 사용 가능
        * 개인 페이지 : 자기 자신만 사용
